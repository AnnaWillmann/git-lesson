• git clone path-to-directory
  • git status - статус текущего репозитория
  • git add path-to-file - добавить новый файл под контроль гита
  • git commit -am "add new file index.html" зафиксировать изменения и дать им комментарий
  • git push - отправляем все закоммиченные изменения в репозиторийна сервер
  • git pull - забрать все обновления с сервера
  • git add . - добавить все новые файлы
